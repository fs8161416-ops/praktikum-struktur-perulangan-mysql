##  Daftar Procedure

1. `tampil_angka()` → angka 1–10
2. `hitung_total()` → total 1–20
3. `bilangan_genap()` → genap 2–20
4. `total_belanja()` → sampai 500000

```sql

CALL tampil_angka();
CALL hitung_total();
CALL bilangan_genap();
CALL total_belanja();
```
